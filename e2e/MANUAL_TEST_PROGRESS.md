## 익명 모드 + 긴 답변 레이아웃 (추가 검증)
- ✅ **익명 모드(E6)**: rooms.mode=anonymous + revealing 로 재현. 공개 카드에 **작성자(from) 미표시** 확인(실명 때 보이던 "from. OO"가 사라짐). 상단 배지 "공개 중 · 익명 모드".
- ✅ **중간 길이 답변**: 긴 문단(공백 포함) 줄바꿈·문단 간격 정상, 카드 내 표시. 공백 없는 한글 초장문도 CJK 줄바꿈으로 가로 오버플로 없음.
- 🐛 **[버그/레이아웃] 긴 답변 클리핑**: ~1920자(쓰기 한도 2000자 내) 답변이 **공개(reveal) 카드 하단에서 잘림 + 스크롤 없음** → 뒷부분 읽기 불가.
  - 위치: RevealView 카드 답변 영역. 컨테이너가 `flex min-h-0 ... overflow-y: hidden` 이라 카드 높이 초과분이 잘림.
  - 제안: 답변 영역에 `overflow-y:auto`(스크롤) 부여 또는 폰트/높이 자동 조정. (관리자 결과 저장 이미지 분할과 별개로 화면 표시 한정 이슈)
  - 참고: `word-break:normal/overflow-wrap:normal` 이라 공백 없는 **라틴** 초장문은 가로 오버플로 가능(한글은 안전).
- 메모: 레이아웃 테스트 위해 시오에게 온 메시지 2건의 content를 길게 수정함(원본은 이번 세션의 테스트 답변 문자열이라 사용자 데이터 아님). assignments/주제는 그대로 유지.

- 🐛→✅ **[수정] 작성(Writing) 화면 긴 주제 클리핑**: 루트가 `h-dvh + overflow-hidden`(페이지 스크롤 없음)인데 주제 폰트가 `text-3xl`로 과대 → 긴 주제 시 textarea·제출 버튼이 화면 밖으로 밀려 **잘리고 누를 수 없었음**.

  - 검증: 807px 높이에서 카드 내부 스크롤로 답변+제출까지 도달 가능 확인.
- ✅ **[정정] 공개(Reveal) 답변 영역은 버그 아님**: 이미 `overflow-y-auto`([RevealView.tsx:120](../src/components/game/RevealView.tsx))로 긴 답변 스크롤됨. 앞서 "클리핑" 진단은 오진(맥OS 오버레이 스크롤바 미표시).

## 작성/공개/종료 결과 (DB assignments 그대로 사용 — start로 재생성 금지)
> ⚠️ 중요: `/api/game/start`는 설계상 시작 시 `pickTopics()`로 assignments를 **새로 생성**(기존 동일 round 삭제). 사용자가 DB에 직접 넣은 주제("내용1/2/3")를 보존하려면 start를 다시 누르면 안 됨. 이번 검증은 DB에 있던 assignments/messages를 그대로 사용.
- ✅ **C5 모드 토글**: 익명(기본)→실명 전환, 상단 배지·선택 반영(broadcast/DB). (모드 변경 리렌더 시 채팅 비워짐 — 채팅 비영속 설계라 정상)
- ✅ **C6 시작 → D 작성**: D1 타이머 2:00·"제출 0/2", D2 작성현황 3명·질문 1/2 대상/주제 표시.
- ✅ **D3 제출**: 답변 입력→제출→질문 2/2 전환, "제출 1/2", 피어 "완료" 실시간. (제출 confirm은 도구상 window.confirm 오버라이드로 처리; 앱은 confirm 사용)
- ✅ **D4 자동 공개 전환**: 전원 완료 → revealing 자동 전환.
- ✅ **E1**: 페이지 = 소개(0) + 답변. "메시지 n/3 장", 페이지 점 인디케이터.
- ✅ **E2 이전/다음**: 카드 양옆 버튼(aria 이전/다음 카드) 동작, 페이지/대상 이동.
- ✅ **E3 연타(연속) 수렴**: 다음 4연타 → 정확히 4칸 뒤로 수렴, 이전 3연타 → 3칸 앞으로 수렴. 되돌아감 없음(디바운스+절대위치).
- ✅ **E4 바로가기**: 참가자 칩 클릭 → 해당 대상 소개로 점프.
- ✅ **E6 실명 표시**: 각 답변에 "from. 작성자" 표시(실명 모드). (익명 모드는 기본값으로 from 미표시 — 사전 확인됨)
- ✅ **E8 종료 전환**: 마지막 대상 끝 장에서 다음 화살표 비활성 + "롤링페이퍼 마침" 버튼 → finished.
- ✅ **E9 공개단계 채팅**: 피어(루디) 공개 단계 채팅 방장 수신.
- ✅ **F1 종료 화면**: "게임 종료" + 대상별 결과 카드(각 답변 2, from 작성자).
- ✅ **F2 결과 모달**: 카드 클릭 → 모달(To.대상, Q.주제, 답변, from, n/2, 닫기 X).
- ✅ **F3 모달 답변 이전/다음**: 다음 → 2/2 이동, 끝에서 화살표 비활성. 닫기 동작.
- ✅ **F4 다시 시작하기**: 방장 전용 버튼 노출 확인(클릭 미수행 — game/reset은 lobby 복귀하며 assignments/messages 보존이지만 상태 변경 피하려 보류).
